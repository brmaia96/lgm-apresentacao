§ 03 · Como publicar

Fluxo de uso.

  1. Abra o arquivo do carrossel no Chrome ou Edge.
  2. Use a extensão GoFullPage ou o atalho F12 → ⋮ → Capture node screenshot sobre cada .slide. Cada slide exporta em 1080×1350 px, pronto pro feed.
  3. Alternativa simples: botão direito no slide → Imprimir → Salvar como PDF, depois fatiar no Canva.
  4. Publique na ordem (capa → conteúdo → CTA). Cole a legenda da §04 abaixo.
  5. Use sempre a mesma fonte no Reels: Gill Sans para título, Fraunces Italic para acento, JetBrains Mono para números.

Carrossel 05 · Anatomia de um projeto
CAPTION · 05O que vem junto com um projeto de casa? Se o "projeto" que te entregaram é só a planta baixa e alguns renders — falta 80 % do trabalho. Um projeto completo tem 7 pranchas: I · Arquitetônico (plantas, cortes, fachadas) II · Estrutural (fundação, formas, armaduras, ART) III · Elétrico (quadros, circuitos, SPDA) IV · Hidrossanitário (água, esgoto, pluviais) V · Memorial descritivo VI · Projeto legal para prefeitura VII · Modelo 3D / render Cada ausência vira retrabalho em obra. Retrabalho vira aditivo. Aditivo vira dor de cabeça. Salve este post. Envie pra quem está contratando projeto. — LGM · Eng.
